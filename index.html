- HTML -

HTML, or HyperText Markup Language, is the foundation of the web. It's a markup language that's used to create the structure and content of web pages. In this section, we'll cover the basics of HTML, including tags, attributes, and elements, as well as how to create a basic HTML document.


Tags and Attributes

HTML tags are used to define the structure and content of a web page. Attributes provide additional information about a tag, such as its class or ID. Here's an example of an HTML tag with some attributes:


          <div class="container" id="main">
            <h1>Welcome to my website!</h1>
            <p>This is some example text.</p>
          </div>

In this example, the div tag defines a container element with the class container and the id main. The h1 tag defines a heading, and the p tag defines a paragraph of text.


Links and Images

HTML can also be used to include links and images on a web page. Here's an example:


          <a href="https://www.example.com">Click here to visit Example.com</a>
          <img src="https://www.example.com/image.jpg" alt="An example image">

In this example, the a tag creates a link to Example.com, and the img tag displays an image with the specified URL and alt text.


Elements

HTML elements are the building blocks of web pages. They are used to define the structure and content of a web page and can be thought of as containers that hold different types of content, such as text, images, and multimedia.

HTML elements are defined using HTML tags, which are enclosed in angle brackets. For example, the <h1> tag is used to define a heading, while the <p> tag is used to define a paragraph.

HTML elements can also have attributes, which provide additional information about the element. For example, the <img> tag is used to define an image element, and it has attributes such as src (which specifies the image source) and alt (which provides alternative text for the image).

In the following sections, we'll explore some of the most common HTML elements in more detail and provide examples of how to use them in your HTML documents.

<h1> - <h6> Headings

Headings are used to define the main headings of a web page. HTML has six levels of headings, with <h1> being the most important and <h6> being the least important.

Example usage:


          <h1>Welcome to my website</h1>
            <h2>About Me</h2>
              <h3>Experience</h3>
                <h4>Web Development</h4>
                  <h5>Design</h5>

<p> Paragraphs

Paragraphs are used to define blocks of text on a web page.

Example usage:


          <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>

<a> Links

Links are used to create clickable hyperlinks that lead to other web pages, documents, or resources.

Example usage:


            <a href="https://developer.mozilla.org/en-US/docs/Web/">MDN Web Docs</a>

<img> Images

Images are used to display images on a web page.

Example usage:


            <img src="dog.jpg" alt="Dog Picture">

<ul> and <li> Unordered Lists

Unordered lists are used to create bulleted lists on a web page using list items <li>.

Example usage:


            <ul>
              <li>Item 1</li>
              <li>Item 2 </li>

            </ul>

Ordered lists are used to create numbered lists on a web page using list items <li>.

Example usage:

<form> Forms

Forms are used to collect user input on a web page, such as text input, radio buttons, checkboxes, and more.

Example usage:

              <input type="email" id="email" name="email"><br><br>

These are just a few examples of the many HTML elements available. By mastering the basics of HTML, you'll be able to create a wide range of web pages and applications.

- CSS -

CSS, or Cascading Style Sheets, is used to style and layout web pages. It's what gives web pages their visual design, including fonts, colors, and layout. In this section, we'll cover the basics of CSS, including selectors, properties, and values, as well as how to apply CSS styles to HTML documents.


CSS Selectors

CSS selectors are used to select and target specific HTML elements on a web page. They allow developers to apply styles to specific parts of a web page, rather than applying styles to the entire page.

There are many different types of CSS selectors, including element selectors, class selectors, ID selectors, and more. In the following sections, we'll explore each type of selector in more detail and provide examples of how to use them in your CSS styles.

Here are some examples of CSS selectors:

  1. Descendant Selector: selects all elements that are descendants of a specified element. 
    2. 
            div p {
              color: red;
            }

    In this example, all <p> elements that are descendants of <div> elements will have a red color.


  2. Class Selector (.): selects all elements with a specified class attribute. 
    3. 
            .my-class {
              background-color: yellow;
            }

    In this example, all elements with the class attribute my-class will have a yellow background color.


  3. ID Selector (#): selects a specific element with a unique ID attribute. 
    4. 
            #my-id {
              font-weight: bold;
            }

    In this example, the element with the ID attribute my-id will have a bold font weight.


  4. Attribute Selector: selects elements based on the presence or value of a specific attribute. 
    5. 
            input[type="text"] {
              border: 2px solid black;
            }

    In this example, all <input> elements with the attribute type set to "text" will have a black solid border of 2px.


    
            * {
              margin: 0;
              padding: 0;
            }

    In this example, all elements on the page will have a margin and padding of 0.

These are just a few examples of CSS selectors. There are many more selectors that can be used to target specific elements on a page.


CSS Properties and Values

CSS properties and values are used to define the visual design of HTML elements on a web page. Properties are used to specify the type of style, such as font size or color, while values are used to define the specific style itself.

CSS provides a wide range of properties and values for developers to use, including text properties, background properties, and layout properties. In the following sections, we'll explore some of the most common CSS properties and values in more detail and provide examples of how to use them in your CSS styles.


Long story short...

CSS is used to style HTML documents. Selectors are used to target specific elements in the HTML, and properties and values are used to define the styling. Here's an example:


        .container  {
          width: 80%;
          margin: 0 auto;
          background-color: #f2f2f2;
        }
        
        h1  {
          font-size: 42px;
          color: blue;
        }

        p  {
          font-size: 20px;
          line-height: 1.5;
        }

In this example, the .container selector targets all elements with the class container and sets the width, margin, and background color. The h1 and p selectors target the heading and paragraph elements, respectively, and set their font size, color, and line height


Layout with Flexbox and Grid

CSS can also be used to create layouts using Flexbox and Grid. Here's an example:


        .container  {
          display: grid;
          grid-template-columns: 1fr 1fr;
          grid-template-rows: auto;
          grid-gap: 20px;
        }
        
        .box  {
          background-color: rgb(100, 150, 20);
          padding: 15px;
        }

In this example, the .container selector sets the display to 'grid' and defines two columns and any number of rows with a 20-pixel gap between them. The .box selector sets the background color and padding for each box.

Layout with Flexbox:

Flexbox is a layout model that allows you to create flexible and responsive layouts in HTML and CSS. It's designed to make it easier to align and distribute content within a container, regardless of the size of the container or the size of the content. With Flexbox, you can create complex layouts with just a few lines of code.

To use Flexbox, you first need to create a container element with the display: flex; property. This turns the container into a flex container, and allows you to use Flexbox properties to control the layout of the container's child elements. Some common Flexbox properties include:

  • flex-direction: specifies the direction in which the flex items are laid out (row, column, row-reverse, or column-reverse).
  • justify-content: specifies how the flex items are distributed along the main axis (center, space-between, space-around, or space-evenly).
  • align-items: specifies how the flex items are aligned along the cross axis (center, flex-start, flex-end, or stretch).

Here's an example of how to use Flexbox to create a simple layout:

HTML


            <div class="container">
              <div class="item">Item 1</div>
              <div class="item">Item 2</div>
              <div class="item">Item 3</div>
            </div>

CSS


          .container  {
            display: flex;
            justify-content: space-around;
          }

          .item  {
            width: 100px;
            height: 100px;
            background-color: beige;
          }

In this example, the container class is set to display: flex;, which turns it into a flex container. The justify-content property is set to space-around, which evenly distributes the child elements within the container with equal spacing between them. The child elements (item) are set to a fixed width and weight and given a background color to make them visible.

Layout with Grid:

Grid is another layout model in HTML and CSS that allows you to create complex layouts with ease. It's designed to work with two-dimensional layouts, meaning you can control both rows and columns in a grid. With Grid, you can create responsive and flexible layouts that adapt to different screen sizes and device types.

To use Grid, you first need to create a container element with the display: grid; property. This turns the container into a grid container, and allows you to use Grid properties to control the layout of the container's child elements. Some common Grid properties include:

  • grid-template-rows: specifies the size and number of rows in the grid.
  • grid-template-columns: specifies the size and number of columns in the grid.
  • grid-gap: specifies the size of the gap between grid items.
  • grid-template-areas: specifies a grid template by referencing named grid areas.

Here's an example of how to use Grid to create a simple layout:

HTML


            <div class="container">
              <div class="item">Item 1</div>
              <div class="item">Item 2</div>
              <div class="item">Item 3</div>
              <div class="item">Item 4</div>
              <div class="item">Item 5</div>
              <div class="item">Item 6</div>
            </div>

CSS


          .container  {
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            grid-template-rows: repeat(2, 1fr);
            grid-gap: 10px;
          }

In the example above, we have a container with six child elements. We want to create a grid with three columns and two rows, so we use the grid-template-columns and >grid-template-rows properties to define the size and number of columns and rows. In this case, we use the repeat() function to specify that we want three columns and two rows, with each column or row taking up an equal amount of space (1fr).

We also use the grid-gap property to specify the amount of space between the grid items. In this case, we set it to 10px.

To position the child elements in the grid, we use the grid-column and grid-row properties. These properties allow you to specify the start and end points of each item in the grid. For example, to position the first item in the top-left corner of the grid, we would use:


          .item:nth-child(1)  {
            grid-column: 1 / 2;
            grid-row:: 1 / 2;
          }

This specifies that the item should start in the first column and first row of the grid, and end in the second column and second row of the grid (since we want it to take up one column and one row). We can use similar rules to position the other items in the grid.

With Grid, you can also create named grid areas and reference them in your layout rules. This can make your code more readable and easier to maintain. Here's an example of how to define named grid areas:


          .container  {
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            grid-template-rows: repeat(2, 1fr);
            grid-template-areas: 
              "header header header"
              "main   main   sidebar"
              "footer footer footer";
          }

          .header  {
            grid-area: header;
          }

          .main  {
            grid-area: main;
          }

          .sidebar  {
            grid-area: sidebar;
          }

          .footer  {
            grid-area: footer;
          }

In this example, we define a grid with three rows and three columns, and define named areas for the header, main content, sidebar, and footer. We then use the grid-area property to position each element in the appropriate area of the grid. This can make it easier to understand and modify the layout of your page.

Responsive Design with Media Queries

CSS can also be used to create responsive designs that adapt to different screen sizes. This is typically done using media queries. Here's an example:


          @media only screen  and (max-width: 600px) {
            .container {
              grid-template-columns: 1fr;
            }
          }
